Name: Mod's display name

State: Publication status

Developping:

  • Completed: No plans to add new content/feature (There's update only to fix bug/error/issue)
  • Completed?: There is possibility to update for improvement/tweaks of content/feature
  • Maybe Completed: Literally, maybe completed
  • Completed now: There is possibility to need to update for tML update
  • Priority: The mod is under development (The mod is given priority to develop)
  • Yes: The mod is under development
  • Pausing: There is possibility to resume to develop anywhen
  • Stopping: Literally, stopping (There is possibility to resume to develop)
  • No: Not completed and no plan to resume to develop

Source Code:

  • in local: I have the mod on hand
  • in old backup: I don't have the mod on hand

Code Restorable:

  • Yes: There's the source code which is able to continue to develop from published latest version
  • Checking: Checking now whether the source code is able to continue to develop from published latest version
  • Maybe able: Maybe there is the source code which is able to continue to develop from published latest version in backup
  • Unknown now: Literally, unknown
  • Maybe can't: Maybe there is NOT the source code which is able to continue to develop from published latest version

Strong Backup:

  • Done: Literally, done
  • Planning: Literally, planning
  • No: Not yet
  • Maybe can't: Maybe can't because there is possibility to have lost the source
